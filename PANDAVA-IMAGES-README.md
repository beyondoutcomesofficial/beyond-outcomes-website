## How to Add Them

### Option A — Upload directly to GitHub

1. Save all 5 images from our chat with these exact names
2. Go to GitHub → your repo → `images/pandava/` folder
3. Upload all 5 images

### Option B — Add to local project then push

1. Save the 5 images with the exact names above
2. Place them in: `beyond-outcomes-website/images/pandava/`
3. Upload to GitHub as part of your next commit
